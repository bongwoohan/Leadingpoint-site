기존 DR은 데이터를 복제해 두고, 장애가 발생했을 때 복구하는 방식이 중심이었습니다. Resilio의 기술을 이용한 리딩포인트의 Active-Active NAS DR은 DR 센터의 NAS를 단순한 대기 자원이 아니라, 평상시에도 활용 가능한 Active Site로 전환합니다.

Resilio의 기술은 블록 단위 차등 동기화, 실시간 변경 감지, UDP 기반 WAN 최적화, 체크섬 재개 전송, 대역폭 제어, 파일 잠금, REST API 기반 중앙 관리를 통해 NAS 간 데이터를 빠르고 안정적으로 동기화합니다. 이를 통해 RPO 5초 미만, RTO 수분 이내의 Active-Active NAS DR 구성을 지원합니다.

Why Active-Active
NAS DR

일반적인 NAS DR은 복제는 가능하지만, DR 센터의 NAS가 평상시에는 충분히 활용되지 못하거나, 장애 시 전환 절차가 길어지는 한계가 있습니다. Resilio는 NAS 간 실시간 동기화를 통해 주센터와 DR 센터를 모두 Active하게 운영할 수 있도록 하며, 장애 발생 시에도 빠르게 전환할 수 있는 구조를 제공합니다.

03

The Difference

리딩포인트의 Active-Active NAS DR의 차별점은 스토리지 어레이 내부 복제가 아니라, NAS 간 데이터 동기화 계층에 있다는 점입니다.

즉, 특정 벤더 NAS에 종속된 복제가 아니라, 다양한 NAS와 서버 환경에서 변경된 데이터만 효율적으로 전송하는 범용 동기화 플랫폼입니다.

01 차등 동기화 변경된 블록만 전송합니다.
02 파일 분할 및 체크섬 재개 대용량 파일도 중단 지점부터 이어서 전송합니다.
03 WAN 최적화 UDP 기반 전송으로 고지연, 패킷 손실 환경에 대응합니다.
04 실시간 변경 감지 파일 변경을 감지하는 즉시 동기화합니다.
05 파일 잠금 양쪽 사이트 간 데이터 정합성을 보장합니다.
06 중앙 관리 작업, 대역폭, 상태를 하나의 콘솔에서 관리할 수 있습니다.
07 REST API 운영 자동화 및 관제 시스템 연동이 가능합니다.
