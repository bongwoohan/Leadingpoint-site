06

S3 Object Lock

Immutable Data Backups

S3 Object Lock은 물리적 에어갭의 대안으로, 암호화와 해시를 통해 데이터를 불변(immutable) 상태로 만들어 랜섬웨어 공격이나 실수로부터 보호합니다. 개별 객체 또는 전체 버킷에 락을 설정하면 시스템 관리자조차 삭제·수정이 불가능하며, Wasabi에서는 추가 비용 없이 백업 데이터에 즉시 접근·복원이 가능합니다.

Why needed

악의적 변조, 컴플라이언스 준수, 인적 오류 방지를 위해 데이터 무결성을 보장하며, 원하는 기간(최대 100년) 동안 수정·삭제를 차단합니다. 외부 침입자가 암호화나 요구금 청구를 시도할 수 없어 장기 보존에 적합합니다.

Data Protection

누구도 변경·삭제 불가한 immutable 백업으로 완전한 데이터 무결성을 보장합니다.

Compliance

규제 요구사항을 충족하며, 추가 비용 없이 무료로 제공됩니다.

Seamless Integration

API로 기존 워크플로우에 통합하여 별도 작업 없이 즉시 적용 가능합니다.

Ransomware 대응

사이버 공격 시 안전 장치로 데이터 복구를 보장하며, 숨겨진 API 비용이 없습니다.

07

Ransomware 대응

Defense · Recovery · Continuity

랜섬웨어 위협이 일상화된 환경에서 Wasabi의 Immutable Bucket과 S3 Object Lock은 종합 백업 전략의 핵심으로, 데이터 안전성과 무결성을 100% 보장합니다. 오프사이트 복사본을 불변(immutable) 상태로 유지하여 악의적 변경·삭제에 저항하며, 지정 보존 기간 동안 완벽한 보호를 제공합니다. 이를 통해 빠른 복구와 규제 준수를 실현하며, 누구도 접근할 수 없는 안전한 백업 환경을 구축할 수 있습니다.
